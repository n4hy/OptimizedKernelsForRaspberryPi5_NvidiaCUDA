# Auxiliary Python tooling for OptMathKernels (OptimizedKernelsForRaspberryPi5_NvidiaCUDA)
#
# The library itself is C++/CUDA built with CMake and has NO Python runtime
# dependency. These packages are for development convenience only:
#   - generating/verifying golden reference vectors against the C++/CUDA kernels
#   - plotting benchmark and profiling output
#   - any Python-side validation tests
#
# Lower bounds match the versions present in the local `optenv` venv.
#
# Create the venv and install:
#   python3 -m venv optenv
#   optenv/bin/python -m pip install -r requirements.txt
#
# ---------------------------------------------------------------------------
# SYSTEM (non-pip) DEPENDENCIES — install with apt, NOT pip
# ---------------------------------------------------------------------------
# The build itself needs these C++/CMake toolchain and Vulkan packages. They
# are OS packages and cannot be installed from PyPI; listed here so this file
# is the single place that records everything the repo needs.
#
#   sudo apt update
#   sudo apt install -y \
#       build-essential cmake git pkg-config \
#       libeigen3-dev \                 # Eigen3 (CPU linear algebra)
#       libvulkan-dev mesa-vulkan-drivers vulkan-tools \  # Vulkan runtime/headers
#       glslang-tools \                 # glslangValidator — compiles .glsl -> .spv (REQUIRED for Vulkan shaders)
#       glslc \                         # shaderc glslc (alternative GLSL->SPIR-V compiler)
#       spirv-tools                     # spirv-val / spirv-opt (SPIR-V validation & optimization)
#
# Without glslang-tools, CMake configures fine but prints
#   "glslangValidator not found. Shaders will not be compiled."
# and the Vulkan backend has no compute shaders to load at runtime.
#
# Optional NVIDIA CUDA support additionally needs cuda-toolkit-12-8+ (see README).
# ---------------------------------------------------------------------------


# Reference numerics & golden-vector generation
numpy>=2.3.5

# Signal-processing references mirroring the kernels
# (FFT, IIR filters, resampling, conv2d, CFAR/CAF, transcendentals)
scipy>=1.16.3

# Benchmark / profiling result plotting
matplotlib>=3.10.7

# Test runner for Python-side validation harnesses
pytest>=9.0.2
